Implementing interfaces

  1. git clone https://github.com/rlindsgaard/interface-example
  2. Implement:
    • nextCommit
    • prepareWorkspace
    • commit
    • rollback
  3. Give it back

nextCommit

  • Reset to latest successful integration
  • Return next commit to be verified

prepareWorkspace

  • Retrieve latest changes
  • Merge next commit with integration branch
  • Make any other necessary preparations before build phase

commit

  • Changes are in the integration branch
  • Push changes to repository

rollback

  • Leave the repository workspace as before prepareWorkspace
